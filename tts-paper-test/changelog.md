# TSS AI测试平台 更新日志

## v1.0.0 (2026-07-15)

### 新增功能

#### 核心测试流程
- **需求分析模块**: 上传需求文档，AI自动分析提取测试点
- **测试点管理**: 完整CRUD，支持批量删除
- **测试方案**: AI自动生成方案，人工审核确认
- **测试用例**: AI生成用例，支持手动创建
- **测试执行**: 选择用例执行，实时进度监控
- **分析报告**: 统计概览，图表可视化

#### WebSocket实时通信
- **实时状态推送**: 测试执行状态实时更新
- **房间订阅**: 支持按执行ID订阅状态
- **断线重连**: WebSocket断开时自动切换到轮询模式
- **通知推送**: 支持全局通知推送

#### 用户权限管理
- **用户管理**: 创建、编辑、删除用户
- **角色控制**: 管理员和普通用户角色
- **密码安全**: MD5哈希存储
- **登录验证**: 数据库验证用户凭证
- **权限装饰器**: 基于角色的访问控制

#### 数据库持久化
- **SQLite数据库**: 使用SQLite替代JSON文件存储
- **14个数据表**: requirements, test_points, test_plans, test_cases, test_executions, agents, mcp_services, skills, prompt_templates, llm_providers, knowledge_patterns, knowledge_bugs, de_ai_strategies, web_scripts
- **数据迁移**: 自动从JSON文件迁移数据
- **查询优化**: 支持分页、筛选、搜索

#### LLM集成
- **多提供商支持**: 支持OpenAI和Anthropic API
- **连接测试**: 验证API密钥和连接
- **需求分析**: 调用LLM分析需求，提取测试点
- **测试方案生成**: 调用LLM生成测试方案
- **测试用例生成**: 调用LLM生成测试用例
- **降级机制**: LLM调用失败时自动使用模拟数据

#### Web自动化测试
- **脚本管理**: 创建、编辑、删除自动化脚本
- **测试步骤**: 支持导航、点击、输入、等待、断言等操作
- **脚本执行**: 实时执行状态监控
- **录制功能**: 模拟录制用户操作

#### 配置管理模块
- **知识库**: 测试模式和Bug知识管理
- **Agent配置**: 多智能体协作配置
- **MCP服务**: 模型上下文协议服务集成
- **Skill配置**: 可复用技能模块
- **提示词工程**: 提示词模板管理
- **大模型配置**: LLM提供商管理
- **去AI味**: 降低AI检测率策略

### 技术特性

- **数据持久化**: SQLite数据库存储，重启后数据不丢失
- **完整CRUD**: 所有模块支持增删改查
- **实时状态**: WebSocket风格的状态轮询
- **响应式设计**: 支持桌面和移动端
- **模块化架构**: 清晰的代码组织

### 文件结构

```
tts-paper-test/
├── web/
│   ├── app.py              # Flask应用 (1000+行)
│   ├── data/               # 数据存储目录
│   ├── static/
│   │   ├── css/style.css   # 样式文件
│   │   └── js/main.js      # JavaScript
│   └── templates/          # 17个HTML模板
├── agents/                 # AI Agent模块
├── core/                   # 核心模块
├── knowledge/              # 知识库
├── prompts/                # 提示词模板
├── tests/                  # 测试代码
├── scripts/                # 工具脚本
├── main.py                 # 主入口
├── pyproject.toml          # 项目配置
└── start.sh                # 启动脚本
```

### 启动方式

```bash
cd tts-paper-test
source .venv/bin/activate
python main.py web
```

访问: http://localhost:5001
默认账户: admin / admin123
