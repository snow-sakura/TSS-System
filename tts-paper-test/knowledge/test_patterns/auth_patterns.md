# 认证测试模式库

## 会话管理测试模式

### 模式：会话创建
- 登录成功→创建会话
- 会话ID唯一性
- 会话超时设置

### 模式：会话维持
- 请求携带会话ID
- 会话有效期检查
- 并发请求处理

### 模式：会话销毁
- 登出→销毁会话
- 超时→自动销毁
- 手动注销→立即销毁

## 权限控制测试模式

### 模式：角色验证
- 管理员→可访问所有功能
- 普通用户→仅可访问授权功能
- 未登录→仅可访问公开功能

### 模式：资源访问控制
- 用户A不能访问用户B的资源
- 管理员可访问所有资源
- 匿名用户仅可访问公开资源

### 模式：API端点保护
- 保护端点需要认证
- 公开端点无需认证
- 管理端点需要管理员权限

## 安全测试模式

### 模式：密码安全
- 密码加密存储
- 密码强度要求
- 密码历史检查（防止重复使用）

### 模式：会话安全
- 会话ID随机生成
- 会话固定攻击防护
- 会话劫持防护

### 模式：CSRF防护
- 表单提交需要CSRF token
- AJAX请求需要CSRF token
- 跨域请求被拒绝

## 并发测试模式

### 模式：并发登录
- 同一用户多处登录
- 登录冲突处理
- 会话互斥

### 模式：并发操作
- 同时修改同一资源
- 资源锁定机制
- 乐观锁/悲观锁
